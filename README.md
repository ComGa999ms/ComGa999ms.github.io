# Personal Web Profile (GitHub Pages)

A one-page personal profile site built with plain HTML, CSS, and JavaScript.

## Project Structure

- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/profile.webp`
- `assets/img/og-cover.webp`
- `WEB_PROFILE_GITHUB_PAGES_PLAN.md`

## Local Preview

Open `index.html` directly in browser, or run with any static server.

Example with VS Code Live Server:
1. Install Live Server extension.
2. Right-click `index.html`.
3. Click **Open with Live Server**.

## Deploy To GitHub Pages

1. Push all files to the `main` branch.
2. Open GitHub repository settings.
3. Go to **Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main`, folder: `/(root)`.
6. Save and wait 1-3 minutes.
7. Open: `https://comga999ms.github.io/`.

## Quick Customize Checklist

- Update email links in `index.html`.
- Replace LinkedIn and YouTube URLs in `index.html`.
- Replace project demo/source links in `index.html`.
- Update avatar and OG image in `assets/img/` if needed.
- Optional: add your CV PDF and change `Request CV` button target.
